# Language Model Work

## Overview
A Python project for language model development and experimentation. Currently serves a placeholder page to confirm the environment is correctly configured.

## Project Structure
- `main.py` - Simple HTTP server that serves a landing page on port 5000
- `.gitignore` - Python-specific ignore patterns

## Running the Project
The server starts automatically via the configured workflow:
```bash
python main.py
```

The server binds to `0.0.0.0:5000` and serves HTTP requests.
